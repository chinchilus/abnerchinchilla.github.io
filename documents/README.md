# Documents

Upload PDF files to the folder that matches the website section. After adding a file, commit and push to GitHub — the site updates in about 1–2 minutes.

**Important:** Use the exact filenames listed below so the Download buttons work. To add a new document, upload the PDF and add a new row to the matching HTML page.

## Folder guide

| Folder | Website page | HTML file to edit when adding new items |

| `insurance/` | Insurance | `insurance.html` |

## How to upload via GitHub

1. Open the repo on GitHub.com
2. Navigate to the correct folder under `documents/`
3. Click **Add file → Upload files**
4. Drag and drop your PDF (use the filename from the list above)
5. Click **Commit changes**
